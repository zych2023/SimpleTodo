# 轻简待办 | 美化增强版
简洁好用的桌面待办清单工具，基于开源项目+AI二次修改优化。
⚠️ 未对macOS进行测试，可能存在错误以及未知bug.

## 原项目说明
本项目基于 [xiajingren/xhznl-todo-list](https://github.com/xiajingren/xhznl-todo-list) 二次开发美化
仅用于个人学习、自用美化，非商业用途。

## ✨ 新增&优化功能
1. 三种主题切换：日间模式 / 夜间深色模式 / 亚克力透明模式
2. 左上角支持自定义编辑个人谏言、座右铭
3. 新增窗口置顶功能，始终悬浮在所有软件上方
4. 优化界面外观、UI样式、交互体验
5. 可以保障开机、重启后位置样式固定不变

## 技术栈
Electron + Vue

## 使用说明
1. 下载 Release 安装包直接安装运行
2. 支持桌面常驻、后台隐藏、窗口置顶
3. 一键切换明暗亚克力主题 
4. 本地保存待办数据，无需联网
